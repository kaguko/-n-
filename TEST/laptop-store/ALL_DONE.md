## ⚡ Quick Access

### 🚀 Chạy Ngay (5 phút)
```bash

cd backend && npm install && node server-fixed.js

# Terminal 2
cd frontend && npm install && npm start

# Browser

http://localhost:5000/api/init-db   # Initialize DB
```
