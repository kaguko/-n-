## вҡЎ Quick Commands

### Start Everything (3 terminals)

**Terminal 1 - Backend:**
```bash
cd backend && npm install && node server-fixed.js
# вң… http://localhost:5000/api/products
```

**Terminal 2 - Frontend:**
```bash
cd frontend && npm install && npm start
# вң… http://localhost:3000
```

**Terminal 3 - Database:**
```bash
# Open browser:
http://localhost:5000/api/init-db
# вң… Database initialized
```

### Useful Commands

```bash
# Development
npm run dev              # Development mode
npm run build            # Build for production
npm run lint             # Check code quality
npm test                 # Run tests

# Database

# Git

# Deployment
```
