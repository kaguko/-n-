## 🚀 Cách Sử Dụng

### Khởi Động Nhanh (5 phút)

```bash
# Terminal 1 - Backend
cd backend
npm install
node server-fixed.js
# ✅ http://localhost:5000

# Terminal 2 - Frontend  
cd frontend
npm install
npm start
# ✅ http://localhost:3000

# Terminal 3 - Database
# Mở browser: http://localhost:5000/api/init-db
# ✅ Database khởi tạo
```

### Đăng Nhập Admin
```
Email: admin@example.com
Password: admin123
URL: http://localhost:3000/admin
```
